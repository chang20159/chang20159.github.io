逻辑运算符&& （内联If ）

可以在JSX中嵌入任何表达式并将其包裹在花括号中，JavaScript逻辑&&运算符就可以放在{}中。

它可以像这样控制元素呈现：Try it on CodePen

在JavaScript中，true && 表达式的结果为表达式的结果，并且 false && 表达式 总是计算为false。

因此，如果条件为真，则&&后面的元素将显示。 如果是false，React会忽略并跳过它。
